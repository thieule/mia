# mia-ai (`mia`)

Lightweight AI assistant framework (Python package). This directory is the **editable install root** for `pip install -e ./agents/core[...]` from the monorepo root, or `pip install -e .[...]` when your shell cwd is `agents/core` (sibling deployments live under `agents/ai-*`).

See repository layout and deployment docs in the parent workspace as applicable.
