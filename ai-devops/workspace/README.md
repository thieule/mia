# Mia DevOps workspace (repo seed)

**Mia DevOps** hỗ trợ **triển khai**, **vận hành**, và **mở rộng công cụ** (MCP + `exec` trên host). Thư mục nanobot: `./workspace` theo `config/config.json`. `memory/`, `cron/`, `sessions/` runtime — xem `../.gitignore`.

| Path | Role |
|------|------|
| **`AGENTS.md`** | Vai trò, ưu tiên, an toàn |
| **`docs/AI_PROJECT_WORKSPACE_SPEC.md`** | Layout multi-project nếu dùng chung với `ai-repo` |
| **`TOOLS.md`** | File, MCP, `exec` |
| **`SOUL.md`**, **`USER.md`** | Giọng và bối cảnh hạ tầng |
| **`admin/`** | Chính sách |
| **`agent/`** | Runbook, bản thảo, test runs |
