# Mia BA workspace (repo seed)

**Mia BA** is the **business-analysis** line (requirements, backlog shaping, process clarity, research). This directory is the nanobot **workspace** (`./workspace` in `config/config.json`). Committed markdown seeds role and tool hints; `memory/`, `cron/`, and `sessions/` are runtime paths (see `../.gitignore`).

| Path | Role |
|------|------|
| **`AGENTS.md`** | Primary role, priorities, boundaries |
| **`docs/AI_PROJECT_WORKSPACE_SPEC.md`** | Spec: Git sync for `ai-repo`, `projects/<slug>/` layout |
| **`TOOLS.md`** | How file/MCP/exec tools are expected to be used here |
| **`SOUL.md`**, **`USER.md`** | Voice and team context |
| **`admin/`** | Human-maintained policy |
| **`agent/`** | BA notes, drafts, workshop outputs, pytest JSON if used |
| **`docs/`** | Workspace references, **`docs/templates/`** |
